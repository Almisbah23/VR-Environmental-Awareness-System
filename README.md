# 🌍 VR Environmental Awareness & Training System

## 📌 Overview
This project is a Virtual Reality (VR) based interactive system developed using Unity and C#. It includes three use cases:

- 🌳 Deforestation Awareness Simulation  
- ❤️ CPR Training Simulation  
- 🌌 Solar System Exploration  

The goal is to provide immersive learning experiences using VR technology.

---

## 🎮 Features

### 🌳 Deforestation
- Tree cutting interaction
- Environmental changes (dark sky, sound loss)
- Drought & hunger scenes

### ❤️ CPR Training
- Real-time BPM calculation
- Compression feedback system
- Pass/Fail evaluation

### 🌌 Solar System
- Interactive planets
- Educational information panels
- Space environment navigation

---

## 🛠️ Technologies Used
- Unity Engine
- C#
- XR Interaction Toolkit
- Meta Quest VR

---

## ▶️ How to Run
1. Open project in Unity
2. Load scene from Assets/Scenes
3. Press Play or build for VR device

---

## 🎯 Objective
To demonstrate real-world problems and learning through immersive VR simulations.
